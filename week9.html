Week 9

Overview

While MSI-based analysis continues in parallel, this week I shifted focus to developing the core RGB change detection model using deep learning. I designed a Hybrid Vision Transformer (ViT) and CNN architecture that leverages transfer learning from pre-trained models to address the challenge of limited training data. The architecture uses a ViT-Base encoder pre-trained on EuroSAT satellite imagery to extract global features from 6-channel image pairs (before and after RGB concatenated), combined with a CNN decoder for precise change mask prediction. To prevent overfitting with our growing but still limited dataset, I implemented a strategic layer-freezing approach that fine-tunes only the final Transformer blocks.

Model Architecture Design

Why Hybrid ViT + CNN?

The architecture combines the strengths of two powerful paradigms:

  • Vision Transformer (ViT): Excels at capturing global context and long-range dependencies across the image, crucial for understanding disaster impacts that span large areas
  • CNN Decoder: Provides precise spatial localization and fine-grained boundary detection, essential for accurate change mask generation
This hybrid approach outperforms pure CNN or pure Transformer architectures for change detection tasks.

Architecture Components:

1. Input Processing - Early Fusion Strategy:

  • Concatenate Before (RGB) and After (RGB) images → 6-channel input
  • Input shape: [Height × Width × 6]
  • Early fusion allows the model to learn joint feature representations from both time periods
  • More effective than late fusion for capturing subtle changes

2. ViT-Base Encoder (Pre-trained on EuroSAT):

  • Pre-training Source: EuroSAT dataset (27,000 labeled Sentinel-2 satellite images)
  • Why EuroSAT: Already trained on satellite imagery, understands remote sensing features
  • Architecture: 12 Transformer layers, 768 hidden dimensions, 12 attention heads
  • Patch Size: 16×16 pixels divided into patches for Transformer processing
  • Output: Sequence of patch embeddings capturing global semantic features

3. Strategic Layer Freezing:

  • Challenge: Limited training data (growing dataset still relatively small)
  • Solution: Freeze early and middle Transformer layers, fine-tune only final blocks
  • Configuration:
    • Freeze: Layers 1-9 (general feature extraction)
    • Fine-tune: Layers 10-12 (task-specific adaptation)
  • Benefits: Prevents overfitting, leverages pre-trained knowledge, adapts to local terrain

4. CNN Decoder with Skip Connections:

  • Multi-scale upsampling pathway
  • Skip connections from encoder preserve spatial details
  • Progressive refinement: coarse to fine-grained predictions
  • Final output: Binary change mask matching input resolution

Technical Implementation(ongoing) Details

Data Preprocessing:

  • Normalization: Scale pixel values to [0, 1] range
  • Standardization: Mean subtraction and variance normalization per channel
  • Resizing: Ensure consistent input dimensions (e.g., 256×256 or 512×512)
  • Augmentation: Random flips, rotations, brightness/contrast adjustments

Training Strategy:

  • Loss Function: Combination of Binary Cross-Entropy and Dice Loss
    • BCE handles pixel-wise classification
    • Dice Loss addresses class imbalance (changed pixels are minority)
  • Optimizer: AdamW with weight decay for regularization
  • Learning Rate: Lower rate (1e-4) for fine-tuning pre-trained layers
  • Batch Size: Adjusted based on GPU memory (typically 8-16)
  • Early Stopping: Monitor validation loss to prevent overfitting

Evaluation Metrics:

  • IoU (Intersection over Union): Primary metric for segmentation quality
  • F1-Score: Harmonic mean of precision and recall
  • Precision: Accuracy of predicted change pixels
  • Recall: Coverage of actual changed areas
  • Pixel Accuracy: Overall classification accuracy

Why This Architecture for Sri Lanka Disasters?

  • Transfer Learning: EuroSAT pre-training provides satellite imagery understanding without requiring massive local datasets
  • Global Context: Transformers capture large-scale patterns (entire flood zones, landslide extents)
  • Precise Boundaries: CNN decoder maintains sharp change boundaries critical for damage assessment
  • Adaptability: Layer freezing allows model to adapt to Sri Lankan terrain and disaster characteristics
  • Scalability: As more data is collected, can progressively unfreeze more layers

Challenges and Solutions

  • Challenge: Limited training data compared to typical deep learning requirements
    • Solution: Transfer learning + layer freezing + aggressive augmentation
