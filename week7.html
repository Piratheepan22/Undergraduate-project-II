Week 7

Overview

This week focused on leveraging the multispectral capabilities of our expanded dataset by computing and analyzing spectral indices for change detection. I explored NDVI (Normalized Difference Vegetation Index) and NDWI (Normalized Difference Water Index) to better understand vegetation and moisture changes in disaster-affected areas. Additionally, I enhanced our annotation methodology to capture both immediate (Before→During) and persistent (Before→After) changes, providing richer ground truth labels for model training.

Detailed Update

Spectral Indices for Change Detection

With the 13-band MSI data collected last week, I can now compute spectral indices that highlight specific environmental characteristics invisible in RGB imagery. These indices are mathematical combinations of spectral bands that enhance certain features while suppressing others.

Key Indices Explored:

  • NDVI (Normalized Difference Vegetation Index):
    • Formula: (NIR - Red) / (NIR + Red)
    • Uses Band 8 (NIR) and Band 4 (Red)
    • Detects vegetation health, coverage, and stress
    • Values range from -1 to +1 (higher = healthier vegetation)
  • NDWI (Normalized Difference Water Index):
    • Formula: (NIR - SWIR) / (NIR + SWIR)
    • Uses Band 8 (NIR) and Band 11 (SWIR 1)
    • Detects water content and moisture levels
    • Effective for flood extent mapping

Delta-Index Change Detection

Rather than analyzing indices in isolation, I computed delta (Δ) values to quantify changes between time periods:

  • Δ(Before→During): Index_during - Index_before = Immediate disaster impact
  • Δ(Before→After): Index_after - Index_before = Persistent post-disaster change

These delta maps highlight areas of significant change, making them useful baseline features for change detection models. For example:

  • Large negative NDVI delta = Vegetation loss (landslide, severe flooding)
  • Large positive NDWI delta = Water accumulation (flooding)

Enhanced Annotation Methodology

Previously, we created single binary masks comparing Before and After images. This week, I upgraded the annotation approach to capture two separate masks from each image triplet:

  1. Before→During Mask:
    • Captures changes occurring during the disaster event
    • Identifies immediate impacts (active flooding, fresh landslides)
    • Useful for rapid damage assessment scenarios
  2. Before→After Mask:
    • Captures persistent changes after the event
    • Includes both immediate damage and secondary effects
    • Better represents final damage extent

This dual annotation approach provides richer training data, allowing models to learn both immediate event detection and longer-term impact assessment. Some changes may only be visible in one timeframe (e.g., temporary flooding in During image but subsided in After image).

Spectral Index Characteristics

Understanding how these indices respond to different land cover types:

  • NDVI Behavior:
    • Healthy vegetation: 0.6 to 0.9
    • Sparse vegetation: 0.2 to 0.5
    • Bare soil / Urban: 0.0 to 0.2
    • Water bodies: Negative values
  • NDWI Behavior:
    • Water bodies: Positive values (>0.3)
    • Vegetation with moisture: 0 to 0.3
    • Built-up / Bare soil: Negative values

Why Indices Improve Change Detection

  • Normalized values: Reduces effects of illumination differences between dates
  • Feature enhancement: Amplifies specific change types (vegetation loss, water gain)
  • Complementary information: NDVI + NDWI together separate different change causes
  • Threshold-friendly: Clear value ranges make simple thresholding effective
  • Physical meaning: Indices relate directly to biophysical properties

Research Papers Reviewed

I studied two key papers to understand index behavior and interpretation:

  • NDWI Paper: Focused on water and moisture detection, explained SWIR band sensitivity to water content, provided interpretation guidelines for different NDWI ranges
  • Spatial Patterns of NDVI: Analyzed NDVI spatial patterns from Sentinel-2, discussed vegetation greenness monitoring, addressed seasonal variation considerations

Key Learnings

  • NDVI and NDWI complement each other—NDVI tracks greenness, NDWI tracks moisture
  • Combining multiple indices helps separate change types (flood vs. landslide vs. seasonal)
  • Delta maps provide quick visual assessment of change magnitude and location
  • Dual temporal masks capture both immediate and persistent changes more completely
  • Band alignment and quality are crucial for reliable index comparisons
  • Indices can serve as both features for ML models and standalone change indicators

Index Computation Examples

Sample code for computing spectral indices:

// NDVI calculation

NDVI = (NIR - Red) / (NIR + Red)

NDVI = (Band8 - Band4) / (Band8 + Band4)


// NDWI calculation

NDWI = (NIR - SWIR) / (NIR + SWIR)

NDWI = (Band8 - Band11) / (Band8 + Band11)


// Delta computation

Δ_NDVI_immediate = NDVI_during - NDVI_before

Δ_NDVI_persistent = NDVI_after - NDVI_before
