Week 11

Overview

Returning to MSI analysis work, this week focused on two parallel objectives: implementing a cloud removal preprocessing pipeline for cloud-contaminated Sentinel-2 optical scenes and expanding the tri-temporal dataset with new flood-prone locations across Sri Lanka. I set up the DSen2-CR (Deep Sentinel-2 Cloud Removal) framework to reconstruct cloud-obscured regions, improving the usability of optical imagery affected by monsoon cloud cover. Simultaneously, I collected new Before/During/After triplets from critical flood zones including the Kelani River basin, Mahaweli River near Trincomalee, and Negombo Lagoon, significantly increasing the geographic diversity of our dataset.

Cloud Removal with DSen2-CR

Why Cloud Removal Matters:

Cloud occlusion is a fundamental challenge in optical satellite remote sensing:

  • Monsoon clouds frequently cover disaster-affected areas during critical time periods
  • Cloud shadows obscure ground features and create false change signals
  • Temporal analysis requires consistent visibility of the same location across time
  • Manual annotation becomes unreliable when ground features are hidden
  • Change detection models struggle with mixed cloud/clear training data
Effective cloud removal is essential for creating a robust, usable dataset.

DSen2-CR Framework:

DSen2-CR (Deep Sentinel-2 Cloud Removal) uses a deep learning approach to reconstruct cloud-obscured regions:

  • Architecture: Residual neural network trained specifically for Sentinel-2 data
  • Inputs: Cloud-contaminated optical image + SAR data for structural guidance
  • Output: Predicted cloud-free optical image
  • Key Feature: SAR (Synthetic Aperture Radar) is weather-independent and penetrates clouds
  • Advantage: Preserves scene structure using SAR guidance during reconstruction

Implementation Process:

  1. Environment Setup:
    • Cloned DSen2-CR repository from GitHub
    • Configured Python environment with required dependencies
    • Downloaded pre-trained model weights
  2. Data Preparation:
    • Selected cloud-contaminated Sentinel-2 scenes from our dataset
    • Generated cloud masks identifying cloudy pixels
    • Acquired corresponding Sentinel-1 SAR data for same locations and dates
  3. Cloud Removal Processing:
    • Fed cloud-contaminated optical + SAR data into DSen2-CR model
    • Generated reconstructed cloud-free predictions
    • Applied brightness enhancement for better visualization
  4. Quality Assessment:
    • Visual comparison: Original vs. Predicted cloud-free
    • Evaluated structural preservation and artifact presence
    • Assessed suitability for downstream annotation and analysis

Cloud Removal Results

Tested the pipeline on three representative cloud-contaminated scenes with varying cloud coverage:

  • Set 1: Moderate cloud coverage (~40%)
    • Successful reconstruction of major features
    • Preserved land-water boundaries
    • Minor artifacts in dense cloud areas
  • Set 2: Heavy cloud coverage (~70%)
    • Good structural preservation using SAR guidance
    • Some texture differences in reconstructed areas
    • Overall improvement in scene interpretability
  • Set 3: Patchy cloud coverage (~50%)
    • Excellent results in partially obscured regions
    • Smooth transitions between clear and reconstructed areas
    • Suitable for annotation and analysis

Dataset Expansion: New Flood Zones

Parallel to cloud removal work, I significantly expanded the dataset with new locations representing diverse flood scenarios across Sri Lanka:

1. Kelani River Flood-Prone Area:

  • Location: Greater Colombo region, Western Province
  • Significance: Major urban flooding corridor affecting densely populated areas
  • Disaster Type: Riverine flooding during heavy monsoon rains
  • Coverage: Urban settlements, agricultural lands, infrastructure
  • Collected: Complete Before/During/After triplet with minimal cloud cover

2. Mahaweli River near Trincomalee:

  • Location: Eastern Province, Trincomalee District
  • Significance: Largest river basin in Sri Lanka, prone to seasonal flooding
  • Disaster Type: Large-scale riverine flooding affecting agricultural zones
  • Coverage: River delta, agricultural lands, rural settlements
  • Collected: High-quality triplet showing clear flood extent changes

3. Negombo Lagoon:

  • Location: Western coastal area, Gampaha District
  • Significance: Coastal lagoon system vulnerable to both flooding and storm surge
  • Disaster Type: Coastal flooding combined with lagoon overflow
  • Coverage: Lagoon margins, fishing communities, wetlands
  • Collected: Triplet capturing complex water dynamics and coastal impacts

Benefits of Dataset Expansion

  • Geographic Diversity: Covers Western, Eastern, and coastal zones
  • Flood Type Variety: Riverine, coastal, and lagoon flooding scenarios
  • Urban-Rural Mix: Includes both densely populated and agricultural areas
  • Model Generalization: More diverse training data improves model robustness
  • Validation Coverage: Better representation of Sri Lanka's flood vulnerability patterns

Integration Strategy

Combining cloud removal and dataset expansion:

  • New triplets prioritized locations with better cloud-free availability
  • Cloud removal applied selectively to "During" images when cloud-free alternatives unavailable
  • Maintained original imagery where possible, used cloud-removed versions as fallback
  • Documented which images underwent cloud removal for transparency in analysis

Observations and Learnings

  • Cloud removal significantly improves visual interpretability of heavily obscured scenes
  • SAR-guided reconstruction preserves structural features better than purely optical methods
  • Quality varies with cloud density—works best for moderate coverage
  • Some texture artifacts appear in dense cloud areas but don't impede change detection
  • New locations show different spectral characteristics, enriching dataset diversity
  • Coastal and lagoon areas present unique challenges (water-land boundaries)
